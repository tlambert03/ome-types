ome-types documentation
=======================

``ome-types`` is a set of autogenerated classes for a pythonic interface
into the `OME data model <http://www.openmicroscopy.org/Schemas/OME/2016-06>`_.

It converts the `ome.xsd schema
<https://www.openmicroscopy.org/Schemas/OME/2016-06/ome.xsd>`_ into a set of
python types.
